# Образовательная платформа

Образовательная платформа для изучения языка Токи Пона с уроками, тестами и отслеживанием прогресса.

## Структура проекта
toki-pona-platform/
├── index.html # Главный HTML файл
├── css/
│ └── styles.css # Стили приложения
├── js/
│ ├── app.js # Главный модуль приложения
│ ├── data.js # Данные уроков и тестов
│ └── quiz.js # Модуль управления тестами
└── README.md # Документация

## Функциональность

### 1. Уроки
- 4 урока по языку Токи Пона
- Теоретический материал с примерами
- Словарь слов с переводами

### 2. Тестирование
- Тесты после каждого урока
- Таймер с ограничением времени
- Разные типы вопросов
- Подсчет баллов

### 3. Отслеживание прогресса
- Сохранение результатов в localStorage
- Отображение статистики
- История пройденных тестов

### 4. Дизайн
- Светлая цветовая схема
- Адаптивный дизайн
- Современный интерфейс
- Удобная навигация

## Использованные технологии

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- JavaScript (ES6+, модульная архитектура)
- LocalStorage для сохранения прогресса
- Font Awesome для иконок
